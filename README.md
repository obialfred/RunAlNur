# House Al Nur - Empire Operating System

The central command center for House Al Nur and all its arms.

## Quick Start

```bash
cd runalnur-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## What is This?

RunAlNur is an AI-powered operational system designed to manage the House Al Nur empire - a modern chaebol-style organization with multiple arms:

| Arm | Focus |
|-----|-------|
| **Nova** | Technology & Hardware |
| **Janna** | Real Estate & Development |
| **Silk** | Luxury E-Commerce |
| **ATW** | Media & Content |
| **OBX Music** | Music & Audio |
| **House** | Holding & Operations |
| **Maison** | Family Office |

## Features

- **Command Center** - Unified dashboard for all arms
- **Project Management** - Track projects across all companies
- **Contact Management** - CRM-lite for contacts per arm
- **AI Manager** - Natural language interface to control everything
- **SOP Management** - Standard operating procedures with Process Street
- **Integrations** - ClickUp, Process Street, HubSpot, Supabase

## Stack

- Next.js 15 + React 19
- Tailwind CSS + shadcn/ui
- Supabase (PostgreSQL)
- ClickUp, Process Street, HubSpot APIs
- Claude/OpenAI for AI features
